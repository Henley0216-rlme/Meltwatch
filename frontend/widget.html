API 调用示例

可通过 HTTP 请求批量调用分析接口：

# 单条分析 POST http://localhost:5001/api/v1/analyze Content-Type: application/json {"text": "你的评价文本"} # 批量分析（最多20条） POST http://localhost:5001/api/v1/batch_analyze Content-Type: application/json {"texts": ["文本1", "文本2", "文本3"]} # 响应格式 { "success": true, "data": { "emotion": { "key": "正面", "label": "满意", "icon": "😊", "category": "positive", "score": 0.9412 }, "all_emotions": [...], "suggestion": "建议..." } }

爬虫数据接入

在爬虫程序中调用 API：

import requests def analyze_reviews(reviews): """批量分析评价""" results = [] # 每批最多20条 batch_size = 20 for i in range(0, len(reviews), batch_size): batch = reviews[i:i+batch_size] response = requests.post( 'http://localhost:5001/api/v1/batch_analyze', json={'texts': batch} ) results.extend(response.json()['data']) return results # 使用示例 reviews = ["评价1", "评价2", "评价3"] results = analyze_reviews(reviews)
